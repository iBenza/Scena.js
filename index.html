What is it?

Libretto.js is an HTML based presentation framework. It's powered by HTML5 and CSS transitions. You will install the browser which supports such techniques.

This is inspired by similar frameworks such as Reveal.js and Deck.js, however, Libretto.js is not for beauty's sake. It can be used in everything situations from formal to casual, by designing theme.
